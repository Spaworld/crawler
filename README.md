# Rails_Template
Barebone Rails 4 template with:

### DB:
Postgres database

### Test Suite:
 * RSpec 3.5
 * Capybara
 * FactoryGirl
 * DatabaseCleaner
 * Faker
 * Shoulda's
 * WhineyValidations _throws validation erros in test.log_
 * SimpleCov

### Development:
 * DotENV
 * Pry
 * Hirb

## Renaming:
Run `rails g rename:app_to New-Name` in order to rename the app
